# ☕ Coffee Machine Simulator

A Python-based command-line application that simulates the working of a coffee vending machine.

## 🚀 Features
- Select drinks: Espresso, Latte, Cappuccino
- Checks availability of ingredients
- Accepts coin-based input
- Calculates and returns change
- Displays machine resource report

## 🛠️ Tech Stack
- Python
- Functions
- Loops & Conditional Logic

## ▶️ How to Run

1. Clone the repo  
2. Run:

```bash
python main.py
